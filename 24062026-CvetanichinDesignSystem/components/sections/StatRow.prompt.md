A quiet proof strip — serif figures with small labels, divided by hairlines. Use right after a hero or before a CTA.

```jsx
<StatRow items={[
  { value: '4',   label: 'Sections' },
  { value: '14',  label: 'Days to launch' },
  { value: '5',   label: 'Ready-to-use tools' },
  { value: '€37', label: 'One-time' },
]} />
```

Variants: `dark` (charcoal, rose-light figures) and `light` (ivory, rose-deep figures). Keep to 3–5 items; labels stay lowercase-ish and short.
