Cvetanichin's primary action control — square editorial corners, uppercase tracked label, restrained hover; use for any call to action.

```jsx
<Button variant="primary" arrow onClick={checkout}>Get instant access</Button>
<Button variant="outline" size="sm">Learn more</Button>
<Button variant="accent" as="a" href="#cta" block>Start now</Button>
```

Variants: `primary` (charcoal, rose-deep wipe on hover), `accent` (solid dusty rose), `outline` (hairline rose that fills), `ghost` (text only). Sizes `sm | md | lg`. `arrow` adds a nudging →. `block` for full width. Never use bold or rounded — elegance is in the tracking and the hairline.
